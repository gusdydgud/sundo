# 풍력단지 건설 예정 위치 기반의 AR 영상제공 앱

- **개발 기간** : `2024/09/02 ~ 09/23` (4주)

- ## 프로젝트 개요
GIS 데이터를 활용해 규제구역을 시각화하고, 특정 구역을 지도 상에서 명확히 표시하여 효율적인 관리가 가능하도록 개발되었습니다. 풍력발전기 모델을 AR로 시뮬레이션하여 사용자가 미리 체험할 수 있는 기능을 제공합니다.

### 팀원의 주요 역할
- **김현용**: 안드로이드 로컬 DB 구축, 지도 API 좌표 지정, 마커 추가 및 수정, 뒤로 가기 종료 기능 등
- **백수민**: 메인 UI 설계 및 사업 관련 기능 구현, 모듈 간 데이터 통신 구현
- **서무경**: 스플래시 화면 및 나침반 UI 구성, 유니티와 안드로이드 통합 작업
- **이진석**: 규제지역 API 활용, 마커 배치 및 지도 화면 UI 설계

- ## 개발 환경
- **운영체제**: Windows, Mac
- **개발 도구**: Unity, Android Studio, Visual Studio Code
- **언어**: C#, Kotlin, Java
- **DBMS**: SQLite
- ## 주요 기능
1. **GIS 기반 규제구역 시각화**
2. **효율적인 마커 관리 시스템**
3. **AR 기반 풍력발전기 시뮬레이션**
4. **오프라인 데이터 처리 및 로컬 DB 관리**
